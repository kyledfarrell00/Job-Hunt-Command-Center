# Job Hunt Command Center

A local Streamlit app for tracking and evaluating job applications with SQLite and pandas.

## Features

- Store job opportunities and descriptions
- Score job fit across multiple categories
- Track application status and follow-ups
- Dashboard for active applications and best-fit roles
- Resume tailoring helper

## Local Setup

```bash
python -m venv .venv
.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```
